A Flutter app for converting currencies in real-time.

## Features

- Real-time exchange rates via **Open Exchange Rates API**
- Simple and clean Flutter UI
- Reliable and precise conversions

## Technologies

- **Flutter**: Cross-platform mobile framework
- **Dart**: Programming language
- **Open Exchange Rates API**: Real-time exchange rates

## Getting Started

```bash
cd flutter_currency_converter
flutter pub get
flutter run
```
