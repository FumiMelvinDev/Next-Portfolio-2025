This app generates inspirational quotes and allows users to share them instantly on social media.

## Features

- Random motivational quote generation
- Social media sharing integration
- Flutter responsive UI

## Technologies

- **Flutter**: Cross-platform UI toolkit
- **Dart**: Programming language
- **Share Plus**: For social media sharing

## Getting Started

```bash
git clone https://github.com/FumiMelvinDev/flutter_quotes_app
cd flutter_quotes_app
flutter pub get
flutter run
```
